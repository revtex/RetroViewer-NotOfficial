# Archived Standalone Tools

This directory contains standalone scripts that have been integrated into **Manager.py**.

## Archived Files

### Meta Editor.v2.py
**Status:** Integrated into Manager.py (Tab 1: Video Metadata)

**Original Purpose:**
- Edit video metadata (Title, Tags, Year, Genre)
- Filter by Tags, Year, Genre
- Export filtered playlists
- Tag/Genre selection dialogs

**Now available in Manager.py:**
- First tab: "Video Metadata"
- All original features preserved
- Better integration with other management tools

---

### ReadFileName.py
**Status:** Integrated into Manager.py (Tab 3: Video Scanner)

**Original Purpose:**
- Scan VideoFiles directory for .mp4 files
- Sync with database (add new, remove deleted)
- Update file_list playlist
- Export to text file

**Now available in Manager.py:**
- Third tab: "Video Scanner"
- Interactive UI with progress logging
- Automatic refresh of Video Metadata tab
- Status bar updates

## Why Archive?
These tools are kept here for:
1. **Backward compatibility** - Users who prefer standalone scripts
2. **Reference** - Understanding original functionality
3. **Rollback** - If issues arise with integrated versions

## Recommendation
Use **Manager.py** for all management tasks. It provides:
- Unified interface with 5 tabs
- Better user experience
- Real-time updates between tools
- Centralized video/playlist/tag/genre/timestamp management
